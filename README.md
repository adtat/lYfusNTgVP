# 项目简介

## 核心功能
- 爬取天气网每日天气数据
- 数据保存至MySQL数据库
- 使用Flask作为后端框架
- 利用ECharts进行数据可视化
- 提供Web访问界面

## 适用场景
- 天气数据监控与分析
- 研究天气变化趋势
- 政府气象部门信息发布
- 企业环境监测与决策支持

## 技术特点
- Python爬虫技术：高效获取并结构化天气数据
- Flask后端框架：轻量级Web应用程序框架
- MySQL数据库：高效数据存储和查询
- ECharts数据可视化：丰富的图表类型和交互特性
- 后台管理界面：美观、简洁的用户体验
